# theme-starter

一套饿了么组件库**自由换肤**、切换**暗黑/明亮**模式的示例模板

## Tech Stack

> vue2 + elementUI + **webpack-theme-color-replacer**

## Project setup

```
npm install
```

### Compiles and hot-reloads for development

```
npm run serve
```

### Compiles and minifies for production

```
npm run build
```

## How to USE

1. 安装核心插件

```
npm i webpack-theme-color-replacer -D
```
